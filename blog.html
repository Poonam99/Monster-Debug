    • How Event loop works in js :

  • The Event Loop has one simple job — to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.A programming structure that continually tests for external events and calls the appropriate routines to handle them.
