    • How Event loop works in js :

  • The Event Loop has one simple job — to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.A programming structure that continually tests for external events and calls the appropriate routines to handle them. An event loop is often the main loop in a program that typically waits for the user to trigger something

    • how many kinds of undefined we get in JavaScript :

  • In JavaScript, the undefined is a primitive type and has only one value “undefined”. A variable that has not been assigned a value is of type undefined . A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value. A function returns undefined if a value was not returned . A variable or an object has an undefined value when no value is assigned before using it.
